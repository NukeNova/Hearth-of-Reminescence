
# Hearth-of-Reminescence


Personal little project on making a broad linux cheat sheet




# Beginning 



For controlling cpu properties:

### [tlp](https://linrunner.de/tlp/)

tlp is configured in its ``/etc/tlp.conf`` directory and is started via

` $ Systemctl start tlp.service `

for java:

### [OpenJDK](https://openjdk.org/index.html)


Packaged in almost every distro's repository

run `java -jar` to execute java executables
